## ğŸ“ Description

This project serves as the capstone evaluation for the **Computer Graphics** course. The challenge was to move beyond simple geometric shapes and model a complex, realistic **Programmer's Setup** using **Blender**, and then bring it to life on the web using **HTML5** and **Three.js**.

The result is a responsive portfolio where users can explore the workspace, interact with objects (like the monitor and TV), and learn about the creator through a seamless 3D interface.
